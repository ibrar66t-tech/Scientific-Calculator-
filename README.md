Professional Scientific Calculator
A professional, web-based scientific calculator built with HTML, JavaScript, and Tailwind CSS. This project is contained in a single HTML file, making it highly portable and easy to deploy.
It features a responsive dark-mode UI, calculation history, memory functions, and a full suite of scientific operations including trigonometric, hyperbolic, and inverse functions.
Features
Responsive Design: Looks and works great on both desktop and mobile devices.
Calculation History: Press the "Hist" button to see a sliding panel of your previous calculations. Click any item to reuse it.
Full Scientific Suite:
Trigonometric functions: sin, cos, tan
Inverse functions (2nd): sin⁻¹, cos⁻¹, tan⁻¹
Hyperbolic functions (hyp): sinh, cosh, tanh
Inverse Hyperbolic (2nd + hyp): asinh, acosh, atanh
Angle Mode: Toggle between Degrees, Radians, and Gradians (Deg, Rad, Grad).
Memory Functions: MC, MR, M+, M-, MS with a persistent "M" indicator.
Advanced Operations: xʸ, y√x, 10ˣ, eˣ, x², x³, √, ³√, n! (factorial).
Constants: π and e available via 2nd + ( and 2nd + ).
Tech Stack
HTML: For the core structure.
Tailwind CSS: For all styling and layout, loaded via a CDN.
JavaScript (ES6+): For all application logic, state management, and DOM manipulation.
How to Use
Clone or Download:
Download the scientific_calculator.html file.
Run:
Simply open the scientific_calculator.html file in any modern web browser.
Deployment
This calculator can be hosted on any static web hosting service. A perfect, free option is GitHub Pages:
Ensure your repository has the scientific_calculator.html file (you can rename it to index.html for a cleaner URL).
Go to your repository's Settings > Pages.
Select the main branch as your source.
Click Save. Your site will be live in a minute or two!
